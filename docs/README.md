# Amazon Sales Dashboard

This repository contains the source code, data, and documentation for the Amazon sales analysis project.

## Folder Structure

- `data/`: Raw and processed datasets.
- `notebooks/`: Jupyter notebooks for data cleaning, analysis, and reporting.
- `scripts/`: Python scripts for automation.
- `reports/`: Final report and visualizations.
- `docs/`: Project documentation.

## How to Run

1. Clone the repository.
2. Install required dependencies: `conda env create -f environment.yml`
3. Run notebooks in order: data cleaning → analysis → reporting.

## Dependencies

- Python 3.12
- pandas, matplotlib, seaborn, jupyter
